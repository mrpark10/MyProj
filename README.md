# AlgoVisualizer

알고리즘 동작을 3D/2D로 시각화하고, 시간·공간 복잡도와 실무 활용처를 보여주는 학습 플랫폼.

- **`MYPROJ/`** — 메인 애플리케이션 (React + TypeScript + Vite). 실행/개발은 이 폴더에서 진행합니다.
- **`algo-visualizer/`** — 초기 병렬 작업본. 참고용으로 남아있으며 현재 개발은 `MYPROJ/`에서 이루어집니다.

## 시작하기

```bash
cd MYPROJ
npm install
npm run dev
```

자세한 스택, 기능, 배포 방법은 [`MYPROJ/README.md`](MYPROJ/README.md)를 참고하세요.
